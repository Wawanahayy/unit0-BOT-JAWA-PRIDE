## Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/Wawanahayy/unit0-BOT-JAWA-PRIDE.git
   ```

   ```bash
   cd unit0-BOT-JAWA-PRIDE
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **private keys:**

   - Create and edit `privateKeys.json`

   **Example `privateKeys.json`:**
   ```json
   [
       "YOUR-PRIVATE-KEY"
   ]
   ```


## Usage

- **Run the BOT:**

  ```bash
  npm start
  ```
